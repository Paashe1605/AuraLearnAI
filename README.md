# AuraLearn: Multi-Agentic Multilingual Education Platform

AuraLearn is a fully agentic educational platform designed to break language barriers in learning. It allows users to search for complex topics in their preferred language, orchestrating multiple AI agents to find the best educational videos, generate localized text and audio summaries, and provide pathways for deep-dive learning via Google's NotebookLM.

Built for the Kaggle 5-Day AI Agents Intensive Vibe Coding Capstone Project.
